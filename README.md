To clone a repository:

```
git clone https://github.com/Ramsky69/Finance-Tracker.git

```
then do
```

cd Finance-Tracker

```

once done, Install the backend dependencies:
```
cd backend
pip install -r requirements.txt
```

after do
```
cd ..
cd frontend
npm install
```

That is all for the basics

-ViteReact-TypeSctript
